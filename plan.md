# 칸반보드 구현 계획서

## 프로젝트 개요
HTML, CSS, JavaScript를 사용한 드래그 앤 드롭 기능이 있는 칸반보드 애플리케이션

## 주요 요구사항
- TO-DO, IN-PROGRESS, DONE 3개의 컬럼 구성
- 카드를 드래그하여 컬럼 간 이동 가능
- HTML, CSS, JS 파일 분리

## 파일 구조
```
day03/
├── index.html    # 메인 HTML 구조
├── style.css     # 스타일링
└── script.js     # 드래그 앤 드롭 로직
```

## 구현 단계

### 1. HTML 구조 (index.html)
**목표**: 칸반보드의 기본 구조 생성

#### 주요 요소
- **컨테이너**: 전체 칸반보드를 감싸는 wrapper
- **보드**: 3개의 컬럼을 포함하는 보드 영역
- **컬럼**: 각각 TO-DO, IN-PROGRESS, DONE을 나타내는 컬럼
  - `data-column` 속성으로 컬럼 식별
  - 헤더: 컬럼 제목 + 카드 개수 표시
  - 카드 컨테이너: 카드들을 담는 영역
- **카드**: 드래그 가능한 작업 카드
  - `draggable="true"` 속성 추가
  - 제목(h3)과 설명(p) 포함
- **컨트롤**: 새 카드 추가 버튼

#### 구현된 기능
- 각 컬럼에 샘플 카드 배치
- 시맨틱 HTML 사용
- 외부 CSS, JS 파일 연결

### 2. CSS 스타일링 (style.css)
**목표**: 시각적으로 매력적이고 직관적인 UI 구현

#### 주요 스타일 요소

##### 레이아웃
- **전체 레이아웃**: Flexbox를 사용한 반응형 디자인
- **보드**: 3개의 컬럼을 가로로 배치 (gap: 20px)
- **컬럼**: `flex: 1`로 동일한 너비 유지

##### 디자인 요소
- **배경**: 그라디언트 배경 (보라색 계열)
- **컬럼**: 
  - 밝은 회색 배경 (#f4f5f7)
  - 라운드 코너 (12px)
  - 그림자 효과
  - 최소 높이 500px
- **카드**:
  - 흰색 배경
  - 라운드 코너 (8px)
  - 호버 효과: 살짝 올라오는 애니메이션
  - 그림자 효과
- **카드 카운트**: 보라색 배지로 표시

##### 드래그 상태 스타일
- `.dragging`: 드래그 중인 카드
  - opacity: 0.5
  - 약간 회전된 효과
- `.drag-over`: 드롭 가능한 컬럼
  - 더 어두운 배경색
  - 점선 테두리

##### 버튼 스타일
- 흰색 배경에 보라색 텍스트
- 호버 시 색상 반전
- 클릭 애니메이션

##### 반응형 디자인
- 768px 이하: 컬럼을 세로로 배치

### 3. JavaScript 기능 (script.js)
**목표**: 드래그 앤 드롭 기능 및 동적 카드 관리

#### 주요 함수

##### 초기화 함수
```javascript
initializeBoard()
```
- 모든 카드에 드래그 이벤트 리스너 등록
- 모든 컬럼에 드롭 이벤트 리스너 등록
- 새 카드 추가 버튼 이벤트 등록
- 카드 개수 초기화

##### 드래그 이벤트 핸들러

**handleDragStart(e)**
- 드래그 시작 시 호출
- 현재 카드를 `draggedCard` 변수에 저장
- 카드에 `.dragging` 클래스 추가
- `dataTransfer` 설정

**handleDragEnd(e)**
- 드래그 종료 시 호출
- `.dragging` 클래스 제거
- 모든 컬럼의 `.drag-over` 클래스 제거

**handleDragOver(e)**
- 드래그 중 컬럼 위에 있을 때 계속 호출
- `preventDefault()` 호출하여 드롭 허용
- `dropEffect`를 'move'로 설정

**handleDragEnter(e)**
- 컬럼에 진입 시 호출
- 컬럼에 `.drag-over` 클래스 추가

**handleDragLeave(e)**
- 컬럼을 벗어날 때 호출
- 컬럼에서 `.drag-over` 클래스 제거

**handleDrop(e)**
- 카드를 드롭할 때 호출
- 이벤트 전파 중단
- 드래그된 카드를 대상 컬럼의 카드 컨테이너에 추가
- 카드 개수 업데이트

##### 카드 관리 함수

**addNewCard()**
- 새 카드 추가 버튼 클릭 시 호출
- `prompt()`로 제목과 설명 입력 받기
- 새 카드 DOM 요소 생성
- 드래그 이벤트 리스너 등록
- TO-DO 컬럼에 추가
- 카드 개수 업데이트

**updateCardCounts()**
- 각 컬럼의 카드 개수 계산
- 헤더의 카드 카운트 배지 업데이트

#### 이벤트 플로우
1. 사용자가 카드를 드래그 시작 → `dragstart` 이벤트
2. 다른 컬럼 위로 이동 → `dragenter`, `dragover` 이벤트
3. 컬럼에 카드 드롭 → `drop` 이벤트
4. 카드가 새 컬럼으로 이동
5. 드래그 종료 → `dragend` 이벤트
6. 카드 개수 업데이트
